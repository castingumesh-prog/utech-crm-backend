# U TECH CRM Backend

Starter backend for the U TECH CRM system.

## Setup
1. Copy `.env.example` to `.env`
2. Update database credentials
3. Run `npm install`
4. Run `npm run dev`

## Main endpoints
- `/api/auth`
- `/api/leads`
- `/api/customers`
- `/api/quotations`
- `/api/work-orders`
- `/api/webhooks/whatsapp`
